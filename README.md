# Blaze Up Portfolio

3D interactive portfolio website built with React Three Fiber (R3F) and React.js

## Tech Stack

- React 19
- Vite
- Three.js & @react-three/fiber
- @react-three/drei
- React Router
- React Hook Form
- Framer Motion
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
